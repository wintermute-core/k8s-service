= Kubernetes Service Helm Chart

== About

Helm chart for deploying applications on Kubernetes with support for:

* Zero-downtime rolling deployments
* Auto scaling (HPA/VPA)
* Configuration and secrets management
* Ingress and service endpoints
* Canary deployments

Based on https://github.com/gruntwork-io/helm-kubernetes-services[gruntwork-io/helm-kubernetes-services] repository.

See link:/charts/k8s-service/README.md[chart documentation] for usage details.
